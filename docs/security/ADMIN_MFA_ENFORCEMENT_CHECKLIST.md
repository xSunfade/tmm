# Admin MFA Enforcement Checklist (TMM)

Use this checklist to prove MFA enforcement for all systems that can access consumer financial data.

## Critical systems

- [ ] Supabase dashboard/admin
- [ ] Hosting provider admin
- [ ] Plaid dashboard/admin
- [ ] GitHub org/repo admin
- [ ] Logging/monitoring admin

## For each system

- [ ] MFA policy is enforced for all users (not optional).
- [ ] List of admin users is exported and stored as evidence.
- [ ] No shared admin accounts are used.
- [ ] Break-glass process is documented and access is limited.
- [ ] Last review date is recorded.
